## Дашборд контроля заявок и звонков

### Задача

Руководителю требовался экран, на котором в течение дня видно:

- сколько контактов поступило;
- сколько контактов передано в отдел продаж;
- кто сколько звонков сделал;
- сколько времени сотрудники провели на линии;
- как отличается активность за день и за неделю;
- как распределяется нагрузка между входящими заявками и холодным прозвоном.

### Решение

Был разработан автообновляемый CRM-дашборд с группировкой по направлениям и сотрудникам.  
Данные агрегируются из CRM и телефонии, после чего выводятся в компактной таблице для ежедневного контроля.

# Публичная демо-версия внутреннего дашборда для контроля заявок и звонков.

Проект показывает:

- количество поступивших контактов за день и неделю;
- количество контактов, переданных в отдел продаж;
- количество звонков по сотрудникам и направлениям;
- общее время разговоров;
- сравнение входящего потока и холодного прозвона;
- сводные итоги по команде;
- автообновляемую таблицу для вывода на монитор руководителя.

В рабочей версии дашборд подключался к Bitrix24 CRM и телефонии.  
Публичная версия использует синтетические данные и не содержит реальных имён сотрудников, ID отделов, CRM-полей, внутренних направлений и бизнес-логики компании.
